환생의 불꽃 종류 선택
숫자 표시에서 너무 작은 숫자는 e 를 써서 표기함
예를 들어 3.14159e-3 = 3.141592 x 10^(-3) = 0.003141592
확률이 너무 낮으면 오류 날 수 있음
\[ p : 독립시행 확률, p_n : 각 옵션의 개수(n)별 확률, x_n : 각 옵션의 개수 확률\] 단 보스 장비의 경우 옵션의 개수는 고정으로 4개이다. \[ x1 = 0.4, x2 = 0.4, x3 = 0.16, x4 = 0.04 \] \[ p = (\Sigma_{k=n}^4 p_k) \times (원하는 등급 이상 확률1) \times ... \times (원하는 등급 이상 확률 n) \] \[ p_k = x_k \times \frac{_{19-n}C_{k - n}}{ _{19}C_k} \]
